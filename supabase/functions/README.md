# Edge Functions

Vacía por ahora. `database.md`: "Edge Functions (solo cuando sea necesario)".

Antes de crear una: ¿esto ya lo resuelve una Database Function/trigger, o una Server Action?
Solo usar Edge Functions cuando se necesite código fuera del ciclo de vida de una request
Next.js (webhooks de terceros, cron jobs, integraciones que Postgres no puede hacer por
sí solo).
